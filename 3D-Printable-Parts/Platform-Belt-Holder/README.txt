# Summary

Y Carriage GT2 belt holder for prusa i3.  
Modified source version to use M3 screws in order to fix GT2 belt to the Y-carriage with no zip ties.  

11/11/2016 Updated: Added .sldprt parts (SW2012)



# Instructions

Mount it as shown on the pictures. You will need 4 M3x10 (or longer) screws.
